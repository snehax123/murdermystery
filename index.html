API Key Setup

This app uses OpenRouter to generate mysteries. You need a free API key — it stays in your browser only and is never sent anywhere except OpenRouter.

1
Visit openrouter.ai and sign up for a free account.
2
Go to Keys in the top menu and click Create Key. Copy it (starts with sk-or-).
3
Paste it below and click Save.

Your key is saved to your browser only (localStorage). Nobody else can see it. You can clear it any time from this screen.
